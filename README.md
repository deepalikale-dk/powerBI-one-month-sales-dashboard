## Project Overview
The **One-Month Sales Dashboard** is an interactive Power BI report created to analyze sales performance for a single month. It provides insights into total sales, commissions, order volume, daily trends, product performance, and regional distribution, helping stakeholders make data-driven business decisions.

---

## Objectives
- Analyze overall sales performance for one month
- Track total revenue, commission, and order count
- Identify daily sales trends and fluctuations
- Compare state-wise sales contribution
- Understand product-wise and category-wise performance

---

## Key Metrics
- **Total Sales:** 2M  
- **Total Commission:** 100K  
- **Total Orders:** 1300  

---

## Dashboard Insights
- Daily sales trend highlights peak and low-performing days
- State-wise waterfall chart shows contribution to total sales
- Product analysis identifies top-selling items such as Office Chairs and Projectors
- Category-wise and segment-wise distribution provides deeper sales insights
- Helps identify demand patterns and regional performance
